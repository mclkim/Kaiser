kaiser
=======
Kaiser PHP framework

Description
-------

Prerequisites
-------
1.Composer

Install
-------
```
mkdir homepage
cd homepage
composer require mclkim/kaiser
```

Installing & Local Development
-------
```
cp vendor/mclkim/kaiser/example .
php -S localhost:8000 -t public/
```

Web brower
-------
```
http://localhost:8000/?hello
```
